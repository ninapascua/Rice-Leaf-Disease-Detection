# Rice Leaf Disease Detection

## Overview
CNN-based system for classifying rice leaf diseases using EfficientNet.

## Features
- CNN image classification
- Grad-CAM visualization
- NLP explanation
- RL optimization

## Project Structure
data/
docs/
scripts/
src/

## Setup
pip install -r requirements.txt

## Dataset
Run:
python data/get_data.py
